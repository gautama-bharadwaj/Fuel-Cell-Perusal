Sl.No. Model Version Date Description Excel sheet model
1. v1.0.1 02/02/2020 This model simulates the operation of a PEM Fuel Cell and estimates the activation, ohmic and concentration losses by taking information from the polarization curve provided in the manufacturer's data sheet of a fuel cell. The user is required to input the information read from the polarization curve Download v1.0.1 excel sheet
2. v1.1.1 14/02/2020 This is an updated version of v1.0.1. The following changes have been made :
  • The exchange current density calculation is made more accurate
  • Activation loss is estimated precisely
  • Concentration loss is updated and estimated precisely
  • The user is able to analyse the losses plot
---
3. v2.0.1 02/02/2020 This model simulates the operation of a PEM Fuel Cell and estimates the activation, ohmic and concentration losses by taking certain user inputs. It does not require the user to have any datasheet and requires general information. It plots the polarization curve, it also gives information about different losses and power output along with their characterstics in form of plots. Download v2.0.1 excel sheet
4. v2.1.1 02/02/2020 This model is an extension of model 2 version 1 wherein the parameters like exchange current density and charge transfer coefficients are calculated intrinsically.This model also gives us information about the different losses that exists and the power output of the cell. Download v2.1.1 excel sheet
5. v2.2.1 14/02/2020 This is the combined and updated version of v2.0.1 and v2.1.1. The following changes have been made :
  • Activation loss is updated and estimated precisely
  • Ionic resistance is intrinsically calculated and varies dynamically
  • Concentration loss plot is updated
