About the models


Model 1


This model helps simulate the working of a PEM Fuel Cell with the help of a polarization curve. The various inputs are taken from the plot. This tool helps the user analyse the various losses. This model can be used to analyse the performance of both stack and cell.

Latest version : v1.1.1

Model 2


This model helps simulate the working of a PEM Fuel Cell without the user haviing to input values from the polarization curve. This model is only applicable for a single cell. It too helps the user analyse the various losses.

Latest version : v2.2.1
